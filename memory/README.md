# Memory

Diretório de memória persistente do projeto archflow.
Deve ser atualizado antes de cada commit para que qualquer agente ou desenvolvedor consiga retomar o trabalho sem perder contexto.

---

## Arquivos

| Arquivo | Conteúdo |
|---------|----------|
| [progress.md](progress.md) | O que foi feito, o que está em andamento, o que vem a seguir |
| [architecture.md](architecture.md) | Mapa de pacotes, fluxo de execução, dependências de sistema |
| [decisions.md](decisions.md) | Decisões técnicas e arquiteturais com justificativa |
| [constraints.md](constraints.md) | O que não deve ser implementado e por quê |
| [preferences.md](preferences.md) | Idioma, estilo de código, fluxo de commits, colaboração |

---

## Regra

**Antes de cada commit de código, atualizar `progress.md`** com:
- o que foi implementado
- status dos testes
- qual é o próximo passo
